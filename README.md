## Description
Marvellous Fans is a fun and colourful quiz app that tests users on their Marvel knowledge. Which tracks their times and scores.


## Features
- Multiple-choice questions about Marvel characters, movies, and comics.
- Picture-based questions to make the quiz.
- Score tracking to motivate users to improve their knowledge.
- Questions review section to help users to improve on their knowledge.

## Usage Examples

<ol>
<li>Fill in the fields to begin quiz</li>
<li>Select and answer all question</li>
<li>Select the review buttons to review answers</li>>
</ol>
